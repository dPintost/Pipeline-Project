# Pipeline-Project
This project focuses on training a model to process customer product reviews

Please run the JuPyteR notebook called "Pipelines_project_Diego_Pintos.ipnb".
The notebook has been pre-run for you, but feel free to run all cells. Total computing time, including model optimization should be no more than 20min (depending on machine specs).

In order to run, please make sure the following packages are installed on your machine
- scikit-learn
- numpy
- pandas
- spacy
